### ส่วนของแผนที่และการแสดงผล (Map & Layers)
- [x] **Base Map & Campus Boundary**: แสดงแผนที่หลักและมีเส้นประแสดงขอบเขตของมหาวิทยาลัย (Campus polygon)
- [x] **เส้นทางเดินรถ (Route Path)**: แสดงเส้นทางที่รถรางวิ่งบนแผนที่
- [x] **จุดจอดรถ (Stops)**: แสดงตำแหน่งป้ายจอดรถต่างๆ บนแผนที่ (รองรับการคลิกได้)
- [x] **ตำแหน่งรถแบบเรียลไทม์ (Live Vehicles)**: แสดงไอคอนรถรางที่กำลังวิ่งอยู่บนแผนที่
  - [x] สามารถคลิกที่ตัวรถบนแผนที่เพื่อเลือก (Select) และให้แผนที่แพน/ซูมตามไปที่รถคันนั้นได้
  - [x] ไอคอนรถที่ถูกเลือกจะมีการเปลี่ยนสถานะเพื่อไฮไลต์ให้เด่นขึ้น (เป็นสีทอง/สถานะ selected)

###  ปุ่มควบคุมการนำทางบนแผนที่ (Map Controls)
ผู้ใช้สามารถควบคุมแผนที่ผ่านปุ่มที่ลอยอยู่บนหน้าจอได้ (มือถืออยู่มุมขวาบน, เดสก์ท็อปอยู่มุมซ้ายล่าง):

- [x] **[ ธีมโหมด ] Theme Toggle**: สลับโหมดสว่าง/มืด (Light/Dark mode) ซึ่งจะปรับสีแผนที่ (Invert colors) และ UI ทั้งหมดตามไปด้วย
- [x] **[ ⛶ ] Fit to Campus**: ปุ่มกดเพื่อจัดตำแหน่งและระยะซูมของแผนที่ให้เห็นพื้นที่มอ. ทั้งหมดพอดีกรอบ
- [x] **[ + ] Zoom In**: ซูมแผนที่เข้า
- [x] **[ - ] Zoom Out**: ซูมแผนที่ออก

### แผงควบคุมและข้อมูลรถ (Vehicle Panel & UI Overlays)
- [x] **Header บอกสถานะ**: แสดงชื่อ "BU Tram" พร้อมจุดไฟกระพริบ "Live Tracking" มุมซ้ายบน
- [x] **แผงแสดงรายชื่อรถ (Vehicle List Panel)**:
  - [x] บน Desktop: เป็นพาเนลด้านขวาบน
  - [x] บน Mobile: เป็น Bottom Sheet ดึงขึ้น/ลงจากด้านล่างหน้าจอ
- [x] **ข้อมูลในบัตรประจำรถแต่ละคัน (Bus Card)**:
  - [x] ชื่อ/ป้ายกำกับของรถ (Label) และเป้าหมายสถานีต่อไป
  - [x] แถบ Progress Bar (พร้อมลูกศร) แสดงเปอร์เซ็นต์ระยะทางระหว่างทางที่วิ่งอยู่
  - [x] เวลาที่คาดว่าจะถึง (ETA) เช่น "~1 นาที", "< 1 นาที", "รอออกตัว"
  - [x] ระยะทางที่เหลือถึงป้ายหน้า (เช่น 1.2 KM หรือ 500 M)
  - [x] สถานะรถ (วิ่งปกติ / หยุดรถ)
- [x] **รายละเอียดแผนการวิ่ง (Detailed Timeline)**:
  - [x] เมื่อกดเลือกเบอร์รถ จะมีปุ่ม "ดูรายละเอียด / ซ่อนรายละเอียด"
  - [x] เมื่อกางดูจะเห็นไทม์ไลน์ระบุพิกัดชัดเจนว่า ออกจากป้ายไหนมา (From) และกำลังจะ ไปป้ายหน้าสุดที่ไหน (To)
- [x] **Autofocus**: แผนที่จะคอยแพนกล้องตามรถคันที่กำลังถูกเลือก (Selected Vehicle) อยู่เสมอ
- [x] **Loading Screen**: มีหน้าจอเบลอโหลดดิ้งพร้อมสปินเนอร์ "Loading GPS data..." ระหว่างรอข้อมูลเริ่มแรก
